## ¿ Cómo usar ?
En la terminal

```
	> git clone https://github.com/anamariasosam/react_instagram_clone.git
	> npm install
	> npm start
```

Si quieres verla con el estilo cambia al branch feature/style
```
	> git checkout feature/style
```
